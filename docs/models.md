# 사용 모델 명세

VideoVoice에서 사용하는 AI 모델들의 상세 사양입니다.

## WhisperX large-v3

| 항목 | 값 |
|------|-----|
| 용도 | Speech-to-Text (STT) |
| 모델 크기 | ~3GB |
| VRAM 사용 | ~4GB |
| 지원 언어 | 99개 (ko, en, ru 포함) |
| 정확도 | WER 5~10% |

### 특징
- 단어 단위 타임스탬프 지원
- 화자 분리 (Diarization) 지원
- faster-whisper 기반 고속 추론

## Qwen3 14B

| 항목 | 값 |
|------|-----|
| 용도 | 문장 번역 |
| 모델 크기 | ~8GB (Q4 양자화) |
| VRAM 사용 | ~10GB |
| 컨텍스트 | 32K 토큰 |
| 런타임 | Ollama |

### 대안 모델

| 모델 | VRAM | 품질 | 속도 |
|------|------|------|------|
| qwen3:8b | ~6GB | ★★★☆☆ | 빠름 |
| qwen3:14b | ~10GB | ★★★★☆ | 보통 |
| qwen3:30b-a3b | ~12GB | ★★★★★ | 느림 |

## XTTS v2

| 항목 | 값 |
|------|-----|
| 용도 | Text-to-Speech (TTS) |
| 모델 크기 | ~2GB |
| VRAM 사용 | ~3GB |
| 지원 언어 | 17개 (ko, en, ru 포함) |
| 음성 클로닝 | 5~15초 참조 오디오 |

### 특징
- Zero-shot 음성 클로닝
- 감정 및 억양 보존
- 다국어 동일 화자 지원

## Gemini (품질 검증)

| 항목 | 값 |
|------|-----|
| 모델 | gemini-2.0-flash |
| 용도 | 번역 품질 평가 |
| API | Google AI Studio |
| 비용 | 무료 티어 제공 |

## 총 VRAM 요구량

동시 실행 시 최대 ~10GB 필요 (RTX 3060 12GB 적합)

> **참고**: 각 단계는 순차 실행되므로 동시에 모든 모델이 로드되지 않습니다.
