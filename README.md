# 🎙️ VideoVoice: AI-Powered Hybrid Video Tools

AI 기반 로컬 비디오 **더빙** 및 **자막 생성** 시스템입니다. 클라우드 API의 지능과 로컬 GPU의 개인정보 보호/가격을 결합한 하이브리드 아키텍처로 최상의 퍼포먼스를 제공합니다.

---

## ⚡ 획기적인 자막 성능 개선 (2026.02)

최근 업데이트를 통해 자막 생성 프로세스가 **100배 이상** 빨라졌습니다. 이제 30분 이상의 장편 영상도 안정적으로 처리할 수 있습니다.

> [!IMPORTANT]
> **Soft Subtitles (1초 완성)** 기술 도입!
> 영상을 다시 인코딩하지 않고 자막 트랙만 추가합니다. 결과물 생성 버튼을 누르자마자 처리가 완료됩니다.

-   **30분 장편 영상 정식 지원**: 번역 타임아웃을 30분으로 연장하여 긴 대본도 끊김 없이 처리합니다.
-   **스마트 번역 엔진**: XML 태그(`<sN>`) 방식을 도입하여 문장 누락 없는 완벽한 자막을 생성합니다.
-   **초고속 인코딩**: 자막 입히기 단계에서 발생하는 무거운 렌더링 과정을 생략하여 CPU/GPU 부하를 최소화했습니다.

## 🚀 영상 길이에 따른 엔진 추천 가이드

최적의 결과를 얻으려면 영상 길이에 맞는 엔진을 선택해 주세요.

### 1. 자막 생성 (Subtitling)
| 영상 길이 | STT (음성 인식) | 번역 엔진 | 특징 |
| :--- | :--- | :--- | :--- |
| **5분 이내** | Local (무료) | Ollama (무료) | 비용 0원, 완전 로컬 처리 가능 |
| **5~15분** | **Groq API** | **Gemini** | **추천!** 가장 빠르고 정확한 조합 |
| **30분+** | Groq / Local | **Gemini** | 장편 영상의 안정적인 처리에 최적화 |

### 2. 비디오 더빙 (Dubbing)
| 영상 길이 | 추천 TTS 엔진 | 특징 |
| :--- | :--- | :--- |
| **3분 이내** | **XTTS (Local)** | 내 목소리 복제(Cloning) 기능 사용 가능 |
| **3분 이상** | **Edge TTS** | 끊김 없는 매우 빠른 합성 속도 |

## 📺 알아두면 좋은 팁

> [!TIP]
> **"미디어를 불러올 수 없습니다"** 에러가 뜨나요?
> Soft Subtitles는 원본 화질을 보존하기 위해 비디오 데이터를 그대로 복사합니다. 만약 원본이 **MKV** 형식이거나 고화질 코덱인 경우 브라우저에서 재생이 안 될 수 있습니다.
> **해결책**: 우측 [다운로드] 버튼으로 파일을 저장한 뒤, **팟플레이어(PotPlayer)나 VLC**로 재생하세요. 자막을 자유롭게 껐다 켤 수 있습니다!

## 🖥️ 실행 방법

### 간편 실행 (권장)
```powershell
# 서버 시작 (기존 프로세스 자동 정리 후 시작)
.\start.bat

# 서버 종료
.\stop.bat
```

### 수동 실행 (개발자용)
```powershell
# 터미널 1: 백엔드
venv\Scripts\python -m uvicorn src.web.main:app --reload --host 0.0.0.0 --port 8000

# 터미널 2: 프론트엔드
cd frontend && npm run dev -- --host
```

### 접속
- 🌐 **웹 UI**: http://localhost:5173
- 🔧 **API 문서**: http://localhost:8000/docs

> [!WARNING]
> 터미널 창이 여러 개 열려 있으면 충돌이 발생할 수 있습니다.
> 문제가 생기면 `stop.bat`으로 모든 프로세스를 정리한 후 `start.bat`으로 다시 시작하세요.
